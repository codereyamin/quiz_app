# Quiz App


## Features

 - Clean & Modern User Interface
 - Attend Quiz
 - Total Score
 - Height Score
 - Correct Ans to green
 - Wrong Ans to red
 - Per quiz show 10 second
 - Select quiz hold screen 2 second
 - Timer count
 - progress bar 
 - Height score storage
